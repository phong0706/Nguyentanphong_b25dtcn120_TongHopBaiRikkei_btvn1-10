1. CAU TRUC CHUONG TRINH
------------------------

Chuong trinh duoc chia thanh cac phan chinh:

A. KHAI BAO VA DINH NGHIA
   - #include: Nhap cac thu vien can thiet
   - #define: Dinh nghia hang so (constants)
   - typedef struct: Dinh nghia cau truc du lieu (Question, Player)

B. CAC HAM TIEN ICH (UTILITY FUNCTIONS)
   - setupConsole(): Set UTF-8 de hien thi tieng Viet
   - initializeRandom(): Khoi tao seed cho ham random
   - isValidAnswer(): Kiem tra dap an hop le
   - normalizeAnswer(): Chuyen dap an sang chu hoa
   - trimString(): Loai bo khoang trang o dau/cuoi chuoi
   - isValidName(): Kiem tra ten hop le

C. CAC HAM GIAO DIEN
   - clearScreen(): Xoa man hinh
   - delay(): Delay (nghi) trong mot khoang thoi gian
   - printTyping(): Hien thi text voi hieu ung typing
   - printBox(): Ve box dep voi border
   - printSeparator(): Ve duong ke ngang
   - printQuestionBox(): Hien thi cau hoi voi box dep
   - printChoice(): Hien thi lua chon dep
   - printResult(): Hien thi ket qua (dung/sai)

D. CAC HAM QUAN LY CAU HOI
   - initializeQuestions(): Khoi tao ngan hang 10 cau hoi
   - shuffleArray(): Tron mang (Fisher-Yates Shuffle)
   - selectRandomQuestions(): Chon ngau nhien cau hoi
   - displayQuestion(): Hien thi cau hoi
   - processQuestion(): Xu ly cau hoi (hien thi, nhan dap an, kiem tra)

E. CAC HAM GAME PLAY
   - playGame(): Choi game va tinh diem
   - playGameMode(): Che do choi game (nhap ten, choi, luu diem)

F. CAC HAM QUAN LY FILE
   - comparePlayers(): So sanh diem de sap xep (dung cho qsort)
   - loadLeaderboard(): Doc bang xep hang tu file
   - saveLeaderboard(): Ghi bang xep hang vao file
   - saveScoreToFile(): Luu diem vao file
   - createSampleData(): Tao du lieu mau

G. CAC HAM HIEN THI
   - displayWelcome(): Hien thi man hinh chao mung
   - displayResult(): Hien thi ket qua
   - displayPerformanceMessage(): Hien thi danh gia ket qua
   - displayGoodbye(): Hien thi thong bao tam biet
   - displayMainMenu(): Hien thi menu chinh
   - displayLeaderboard(): Hien thi bang xep hang (co phan trang)

H. HAM MAIN
   - main(): Diem bat dau cua chuong trinh


2. CAC KHAI NIEM CAN BIET
--------------------------

A. STRUCT (CAU TRUC DU LIEU)
   - Struct la kieu du lieu tu dinh nghia, gom nhieu bien thanh mot nhom
   - Vi du:
     typedef struct {
         char name[50];
         int score;
     } Player;
   
   - Su dung:
     Player p;
     p.name = "Nguyen Van A";
     p.score = 10;

B. CON TRO (POINTER)
   - Con tro la bien luu dia chi cua bien khac
   - char *ptr: Con tro den bien kieu char
   - *ptr: Lay GIA TRI cua bien ma con tro tro toi
   - &ptr: Lay DIA CHI cua bien
   
   - Vi du:
     int x = 10;
     int *p = &x;  // p tro den x
     *p = 20;      // Thay doi gia tri cua x qua con tro

C. MANG (ARRAY)
   - Mang la tap hop nhieu phan tu cung kieu
   - int arr[10]: Mang 10 phan tu kieu int
   - arr[0]: Phan tu dau tien (chi so bat dau tu 0)
   - arr[9]: Phan tu cuoi cung

D. FILE I/O (NHAP/XUAT FILE)
   - fopen(): Mo file (r = doc, w = ghi, a = them vao cuoi)
   - fclose(): Dong file (QUAN TRONG: phai dong sau khi dung)
   - fgets(): Doc mot dong tu file
   - fprintf(): Ghi vao file (giong printf)
   - fscanf(): Doc tu file (giong scanf)

E. RANDOM (SO NGAU NHIEN)
   - rand(): Tao so ngau nhien (0 den RAND_MAX)
   - srand(): Khoi tao seed cho rand()
   - time(NULL): Lay thoi gian hien tai (dung lam seed)
   - Vi du: srand(time(NULL)); de moi lan chay co so khac nhau

F. STRING (CHUOI)
   - Chuoi la mang ky tu ket thuc bang '\0'
   - strlen(): Do dai chuoi
   - strcpy(): Copy chuoi
   - strcmp(): So sanh chuoi


3. THUAT TOAN QUAN TRONG
-------------------------

A. FISHER-YATES SHUFFLE (Tron mang ngau nhien)
   - Thuat toan tron mang dam bao moi hoan vi deu co xac suat bang nhau
   - Cach hoat dong:
     1. Bat dau tu cuoi mang (i = n-1)
     2. Chon ngau nhien mot vi tri j tu 0 den i
     3. Doi cho phan tu o vi tri i va j
     4. Giam i xuong 1 va lap lai

B. QUICK SORT (qsort)
   - Thuat toan sap xep nhanh (co san trong C)
   - Can ham so sanh de biet cach sap xep
   - Ham so sanh tra ve:
     - So am: a dung truoc b
     - So duong: a dung sau b
     - 0: a bang b


4. QUY TRINH HOAT DONG
-----------------------

A. KHOI TAO
   1. setupConsole(): Set UTF-8 de hien thi tieng Viet
   2. initializeRandom(): Khoi tao seed cho random
   3. initializeQuestions(): Khoi tao 10 cau hoi

B. MENU CHINH
   1. Hien thi menu: 1. Choi game, 2. Xem BXH, 3. Thoat
   2. Nhan lua chon tu nguoi choi
   3. Thuc hien hanh dong tuong ung

C. CHOI GAME
   1. Nhap ten nguoi choi (co validation)
   2. Chon ngau nhien 10 cau hoi (shuffle)
   3. Duyet qua tung cau hoi:
      - Hien thi cau hoi
      - Nhan dap an (co validation chat che)
      - Kiem tra dung/sai
      - Cap nhat diem
   4. Hien thi ket qua
   5. Luu diem vao file

D. BANG XEP HANG
   1. Doc du lieu tu file
   2. Sap xep theo diem giam dan (qsort)
   3. Hien thi voi phan trang (10 nguoi/trang)
   4. Co badge TOP 1, TOP 2, TOP 3


5. VALIDATION (KIEM TRA HOP LE)
--------------------------------

A. VALIDATION TEN NGUOI CHOI
   - Khong duoc rong
   - Khong qua 50 ky tu
   - Khong chua ky tu dac biet: / \ : * ? " < > |

B. VALIDATION DAP AN
   - Khong duoc de trong (chi nhap Enter)
   - Khong duoc chi co khoang trang
   - Chi duoc nhap 1 ky tu
   - Phai la A, B, C, hoac D (khong phan biet hoa/thuong)

8. LUU Y QUAN TRONG
-------------------

A. CON TRO
   - Phai kiem tra NULL truoc khi su dung
   - Phai khoi tao truoc khi su dung

B. FILE I/O
   - LUON dong file sau khi dung (fclose)
   - Kiem tra NULL sau khi mo file

C. MANG
   - Chi so bat dau tu 0
   - Khong vuot qua gioi han mang (buffer overflow)

D. STRING
   - Luu y ky tu ket thuc '\0'
   - strlen() khong tinh '\0'
